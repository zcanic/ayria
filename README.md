`ayria` is intended to become a local-first desktop presence system rather than a plain chat application.
The repository structure and handoff documents are designed to keep future agents aligned with that architecture.

Current runtime default:
- scaffold mode is still the default
- a real Ollama-backed local inference path now exists for live-mode integration work
